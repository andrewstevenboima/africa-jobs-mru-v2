# 🌊 Africa Jobs — Mano River Union

**The MRU's dedicated youth employment platform**, connecting young talent with employers across Sierra Leone 🇸🇱, Liberia 🇱🇷, Guinea 🇬🇳 and Côte d'Ivoire 🇨🇮.

🔗 **Live site:** [https://YOUR-USERNAME.github.io/africa-jobs-mru](https://YOUR-USERNAME.github.io/africa-jobs-mru)

---

## ✨ Features

- 🗺️ **Interactive MRU Map** — click any country to explore jobs by location
- 💼 **Job Listings** — district/county/prefecture level across all 4 MRU nations
- 🌊 **Mano Points System** — earn MPs through Tributary → River → Delta → Ocean tiers
- 🤝 **Confluence** — Africa Jobs' unique take on following/connecting with members
- 🌍 **Community Feed** — post updates, ask questions, share milestones
- 🎯 **ATS Resume Analyzer** — AI-powered CV scoring with file upload
- 📊 **Statistics** — verified World Bank 2024 data for all 4 MRU countries
- ♿ **Accessibility** — WCAG 2.1 AA, screen reader support, Read Aloud (TTS), dyslexia font
- 🔊 **Text-to-Speech** — full page read-aloud with pause/stop controls
- 🌐 **Multilingual** — English, Français, Krio, العربية, Português
- 🏆 **SDG Aligned** — Goals 4, 5, 8, 10, 17

## 🚀 Getting Started (Local)

Just open `index.html` in any modern browser — no build step, no dependencies.

```bash
git clone https://github.com/YOUR-USERNAME/africa-jobs-mru.git
cd africa-jobs-mru
open index.html   # macOS
# or
start index.html  # Windows
```
